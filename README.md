# Fitur-Redo-dan-Undo-pada-Text-Editor
Program ini adalah fitur pada teks editor dimana akan melakukan undo dan redo setelah kita memasukkan kalimat. Program akan mengundo ketika kita memasukkan perintah "Undo" dan akan meredo teks ketika kita memasukkan perintah "Redo". Program ini juga dilengkapi dengan fitur - fitur lain seperti "tambah", "hapus", "dan "exit"
